# BookStoreX
Simple book store web api
## Net.core WebApi
### Used technologies: 
  * ASP.NET core 3.1,
  * entity framework core 3.1.3,
  * linq,
  * AutoMapper,
  * FluentValidation
  * ms sql server,
  * jwt bearer role based auth</br>
